Hotkeys

```Python
# Selection
ctrl + left arrow : multiple cursor
ctrl + d: expand selection
shift + right arrow: multiple cursor 

# Line
ctrl + shift + up/down: shift line up and down

# Indentation
reindent

# Goto anything
ctrl + p: go to a file
ctrl + p -> @: go to a function
ctrl + p -> #: go to a word
ctrl + p -> :17 : go to line 17

# Command Palette
ctrl + shift + p: open command palette

# Install packages
ctrl + shift + p -> install package

# Snipppets

```
